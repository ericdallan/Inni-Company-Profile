Bagaimana Cara Kerjanya?

Desktop App, Bukan Website Biasa

inni Akun Digi & POS adalah aplikasi desktop seperti Microsoft Word — install sekali, pakai selamanya, 100% offline

1

Download Installer

Setelah pembayaran, terima link download file .exe + license key via email

2

Install di Windows

Double-click installer → Next → Next → Finish. Seperti install software biasa, 5-10 menit selesai

3

Langsung Pakai!

Buka dari desktop shortcut, input license key, dan mulai kelola keuangan — 100% offline!

Data 100% Lokal

Semua data tersimpan di komputer Anda, tidak ada yang dikirim ke cloud.

Tidak Butuh Internet

Cocok untuk area dengan internet tidak stabil atau kantor tanpa WiFi.

Pakai Selamanya

Bayar sekali, tidak ada biaya bulanan/tahunan. Update gratis & opsional.

Multi-Perangkat

1 lisensi untuk 1 PC utama. Perangkat lain dalam jaringan yang sama dapat mengakses melalui file server yang dijalankan di PC utama, dengan konfigurasi awal yang mudah.

License Protection

License terikat Machine ID. Admin bantu transfer license saat ganti PC.

Backup Mudah

1-click backup ke file .sql. Restore kapan saja tanpa ribet.

Spesifikasi Sistem

Spesifikasi Minimal

  • OS: Windows 7/8/10/11 (32/64-bit)
  • RAM: 2 GB
  • Processor: Dual Core
  • Storage: 500 MB free

Spesifikasi Rekomendasi

  • OS: Windows 10/11 (64-bit)
  • RAM: 4 GB+
  • Processor: Core i3/i5 / Ryzen
  • Storage: SSD

PHP & database built-in — tidak perlu install XAMPP
